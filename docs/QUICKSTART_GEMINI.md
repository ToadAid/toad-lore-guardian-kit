# Gemini Gem Quick Start

## Create the Gem

1. Open Gemini and create a new Gem.
2. Suggested name: **Study the Lore — Corpus v1**.
3. Paste the complete contents of [`../SYSTEM_PROMPT.md`](../SYSTEM_PROMPT.md) into **Instructions**.
4. Keep **Default tool** set to **No default tool** for the first closed-book test.

## Add knowledge

Upload all 10 files from [`../platforms/gemini-gem`](../platforms/gemini-gem):

1. `01-foundation-source-records.json`
2. `02-lore-scrolls-early.json`
3. `03-lore-scrolls-later.json`
4. `04-community-commentary-early.json`
5. `05-community-commentary-later.json`

9. `09-codex-reflections-later.json`

Upload the extracted files themselves, not a ZIP archive.

## Save and test

Save the Gem and start a fresh conversation. Test with:

```text
What is Taboshi?
```

```text
Quick study: What is Rune3?
```

```text
Separate what Toadgod directly stated from what the archive interprets about Epoch 3.
```

```text
What happened to the proposed Patience Vault? Distinguish the original proposal from its later status.
```

The last test checks whether the Gem respects later correction and cancellation records rather than treating every historical proposal as currently active.

## Public sharing

Before sharing a Gem publicly, review several answers for provenance discipline. The Gem should distinguish direct source statements from community commentary and interpretation, avoid financial advice, and never claim that ToadAid owns or officially controls Tobyworld.
