# ChatGPT GPT Knowledge Files

Upload all 10 generated JSONL/JSON files in this folder to a ChatGPT GPT's **Knowledge** section.

Paste the repository's [`GPT_INSTRUCTIONS_PROMPT.md`](../../GPT_INSTRUCTIONS_PROMPT.md) into the GPT's **Instructions** field.

The compact GPT prompt is self-contained and designed for the GPT Instructions character limit. Do not combine it with the Gemini prompt or upload the Gemini prompt as an additional knowledge file.

Full setup guide: [`../../docs/QUICKSTART_CHATGPT.md`](../../docs/QUICKSTART_CHATGPT.md)
